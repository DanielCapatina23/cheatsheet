Document Summary

Tag Name Desciption
<html>> HTML tag The tag shows up at the beginning and end of an HTML document. Indicates that the page is written in HTML5
<title> Title Title for the page. Useful for both search angines and users. Showing up in the browser title bar
<head> Head Contains information about the specific page, including title tags, meta data and links to scripts and style sheets
<body> Body Include all content that will be shows to users

Tag Name Description

<tbody> Table Body The set of rows containing actual table data.

<colspan> Column Span Defines how many columns a td element should span.
